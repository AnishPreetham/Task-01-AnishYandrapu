# Project 1 - Rule Based AI Chatbot

## Description

This project is a simple rule-based AI chatbot developed using Python. The chatbot responds to predefined user inputs using a dictionary-based approach and provides appropriate responses.

## Features

* Responds to greetings and common questions
* Handles user input using predefined rules
* Supports an exit command
* Uses input normalization with `strip()` and `lower()`
* Provides a default response for unknown inputs

## Sample Inputs

* hello
* hi
* how are you?
* what is ai?
* who are you?
* thank you
* bye
* exit
